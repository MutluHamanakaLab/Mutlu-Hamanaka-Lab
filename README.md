# Mutlu–Hamanaka Lab GitHub Welcome Repository
Welcome to the Mutlu–Hamanaka Lab GitHub organization.
This repository serves as the landing page for our computational and bioinformatics work, providing an overview of our research areas and links to our published code.

# About the Lab
The Mutlu–Hamanaka Lab in the Department of Medicine, Section of Pulmonary and Critical Care Medicine investigates mechanisms of lung injury, fibrosis, and metabolic regulation.
Our computational efforts include transcriptomics, single‑cell analysis, metabolic pathway modeling, and machine‑learning approaches relevant to pulmonary biology.

# Published Repositories
## Transfer_Learning_Enabled_DTI_Prediction
- A machine‑learning framework for drug–target interaction prediction using transfer learning.
- Repository: https://github.com/MutluHamanakaLab/Transfer_Learning_Enabled_DTI_Prediction

## MTHFD2
- Analysis code for the study “Mitochondrial One‑Carbon Metabolism is Required for TGF‑β‑Induced Glycine Synthesis and Fibrotic Responses.”  
- Repository: https://github.com/MutluHamanakaLab/MTHFD2

## Arginine
- Analysis code for the study “Arginine Promotes the Activation of Human Lung Fibroblasts Independent of Its Metabolism.”  
- Repository: https://github.com/MutluHamanakaLab/Arginine

## ATF4_mTOR
- Analysis code for the study “mTOR signaling regulates multiple metabolic pathways in human lung fibroblasts after TGF‑β and in pulmonary fibrosis.”
- Repository: https://github.com/MutluHamanakaLab/ATF4_mTOR
